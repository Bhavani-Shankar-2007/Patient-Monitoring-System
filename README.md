# 🕷️ SpiderAdhere - Patient Adherence Monitoring Platform

![SpiderAdhere](https://img.shields.io/badge/Theme-Spider--Man%20Red-black?style=for-the-badge&logo=spider-man)
![Spring Boot](https://img.shields.io/badge/Spring%20Boot-3.3.4-brightgreen?style=for-the-badge&logo=springboot)
![Java](https://img.shields.io/badge/Java-21-orange?style=for-the-badge&logo=openjdk)

A fun and powerful **Patient Adherence Monitoring System** with a unique **Spider-Man themed** frontend. Built for the SNS Hackathon.

> "Web slinging your way to better medication adherence!" 🕸️

---

## ✨ Features

- **Real-time Patient Dashboard** with adherence tracking
- **Interactive Patient Selection** with progress visualization
- **Live Data Logging** (Medication, Appointments, Lifestyle Score)
- **AI-Powered Intervention Strategies** using Google Gemini
- **Beautiful Dark Spider-Man Theme** with red accents and web animations
- **Responsive & Modern UI**

---

## 🎨 Theme

**"SpiderAdhere"** – A dark cyberpunk Spider-Man inspired healthcare platform.

- Deep red & black color scheme
- Spider web background effects
- Fun superhero references ("Web slinging complete!", "🕷️")
- Progress bars with flame gradient

---

## 🛠️ Tech Stack

- **Backend**: Spring Boot 3.3.4 + Java 21
- **Frontend**: HTML, CSS, JavaScript (Vanilla)
- **AI**: Google Gemini 2.0 Flash
- **JSON Processing**: Gson
- **Build Tool**: Maven
